# Konjo Boot

Run at session start:

1. Read `PLAN.md` and `KORE_PLAN.md` (if they exist) — understand current objectives
2. Run `git log --oneline -10` — see recent history
3. Run `git status` — check working tree state
4. Read `KONJO_QUALITY_FRAMEWORK.md` — refresh quality standards
5. Read `KONJO_PROMPT.md` — load the adversarial reviewer persona

You are now oriented. Begin work with the Konjo quality mindset active.
